## Required modules/libraries

- bs4
- beautifulsoup4
- motuclient
- pandas
- requests
- siphon
- wget
- xarray

Install via pip:

```sh
pip install -r requirements.txt
```

## Usage

Example:
```sh
python main.py --year=2019 --minutes=30
```
both arguments are required:

>year: the year to download AIS-data.

>minutes: is the subsampling interval in minutes.
